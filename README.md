# 🚗 Ford Car Price Prediction

## 📖 Overview

This project focuses on predicting Ford car prices using Machine Learning techniques. The objective is to analyze vehicle characteristics and build a predictive model capable of estimating car prices accurately.

## 🎯 Problem Statement

Car prices are influenced by multiple factors such as vehicle age, mileage, engine size, transmission type, and fuel type. This project aims to identify these relationships and develop a regression model for price prediction.

## 📊 Dataset Features

* 🚘 Model
* 📅 Year
* ⚙️ Transmission
* 🛣️ Mileage
* ⛽ Fuel Type
* 💰 Tax
* 📈 MPG
* 🔧 Engine Size
* 🎯 Price (Target Variable)
